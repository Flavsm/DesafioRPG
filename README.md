# DesafioRPG


# Setup
Install Java 17 SDK
***

Swagger endpoint

`/swagger-ui/index.html`

H2 DB

`localhost8080/h2`

# Jogar

Alguns Heroes e Monsters foram pre-carregados para a DB.

Endpoints Heroes

`GET /api/heroes/list`

`GET /api/heroes/{id}`

`POST/PUT/DELETE /api/heroes`

Endpoints Monsters


`GET /api/monsters/list`

`GET /api/monsters/{id}`

`POST/PUT/DELETE /api/monsters`

Endpoints Battle


`GET /api/battle/list`

`GET /api/battle/{id}`

`POST/PUT /api/battle`
